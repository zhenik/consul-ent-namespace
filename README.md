# Consul - namespaces

To install consul 1.8.4 ent version
```text
make install
```

To run consul dev mode
```text
make consul
```

To run playbook
```text
make tun
```

## Requirements
- [ansible](https://www.ansible.com/)
- [jq](https://stedolan.github.io/jq/)
- [curl](https://curl.haxx.se/)
